# Arch-DotFiles
My arch linux dotfiles! This repository includes my dwm config, .config files, programs I use, and most importantly my wallpaper.

### Programs I use

*xbindkeys* - I use this to easily make key bindings (You could use dwm config file, but if you use multiple window managers this makes it easy to switch between and keep key bindings).

*zathura* - document reader, I also use zaread for reading school documents, like doc and docx (so much faster than opening libreoffice).

*pywal* - themes basically everything (I need to fix alacritty so that it's background syncs with pywal).

*firefox* - web browser, pywal fetch extension for pywal colors, you can also go into firefox settings and force the background color for websites.

*neovim* - text editor, I still use vim-plug because I'm too lazy to switch to anything else

*fish* - I use fish as my user shell because it's easy and has the features I need, I do use bash for scripts

*ranger* - terminal file browser, I think there are faster options I've seen, but I got ranger setup the way I like it and don't plan on changing

*cmus* - easy to use music player
